**FakeStore implementation (define inline in game_notifier_test.dart or in a shared helper):**

ObjectBox `Store` cannot be easily mocked. The cleanest approach for unit testing is to create a minimal `FakeStore` class. However, ObjectBox `Store` is a native-backed class that's hard to mock. Instead:

Use an **in-memory ObjectBox store** via `openStore()` with a unique temp directory per test. This is the approach recommended by ObjectBox docs for testing — it's the same real ObjectBox API, just with a temp directory that's deleted after the test:

```dart
import 'dart:io';
import 'package:objectbox/objectbox.dart';
import 'package:path/path.dart' as path;
// import mobile project objectbox.g.dart for openStore()

final tempDir = Directory(path.join(Directory.systemTemp.path, 'obx_test_${DateTime.now().microsecondsSinceEpoch}'));
tempDir.createSync();
final store = openStore(directory: tempDir.path);
addTearDown(() { store.close(); tempDir.deleteSync(recursive: true); });
```
