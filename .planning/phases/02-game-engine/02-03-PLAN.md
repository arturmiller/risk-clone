  <name>Task 1: Turn execution engine with phase transitions, elimination, and victory</name>
  <files>risk/engine/turn.py, tests/test_turn.py</files>
  <behavior>
    - execute_reinforce_phase: handles forced card trade (5+ cards), calculates reinforcements (base + continent + trade bonus), calls agent.choose_reinforcement_placement, validates placements sum to total armies, applies placements to state
    - execute_attack_phase: loops agent.choose_attack until None (skip), executes each attack, handles conquest (set conquered flag), handles elimination (mark dead, transfer cards, forced trade if 5+), checks victory after each conquest, draws 1 card at phase end if conquered_this_turn
    - execute_fortify_phase: calls agent.choose_fortify, executes if not None
    - execute_turn: runs reinforce -> attack -> fortify, advances to next alive player, increments turn_number, resets conquered_this_turn
    - check_victory: returns winning player_index if one player owns all territories, else None
    - next_alive_player: skips eliminated (is_alive=False) players
    - Elimination: when a player loses their last territory, mark is_alive=False, transfer cards to eliminator, if eliminator has 5+ cards force immediate trade loop
    - Victory detected immediately after elimination leaves only 1 alive player
  </behavior>
  <action>
    Create risk/engine/turn.py:

    - check_victory(state: GameState) -> int | None:
      Get set of owners from state.territories. If only 1 owner and that player is_alive, return their index. Else None.

    - check_elimination(state: GameState, player_index: int) -> bool:
      Return True if player owns 0 territories.

    - transfer_cards(state: GameState, from_player: int, to_player: int) -> GameState:
      Move all cards from eliminated player to eliminator. Return new state.

    - force_trade_loop(state: GameState, player_index: int, agent: PlayerAgent) -> tuple[GameState, int]:
      While player has 5+ cards: call agent.choose_card_trade(forced=True), execute_trade, accumulate bonus armies. Return (state, total_bonus_armies).

    - execute_reinforce_phase(state: GameState, map_graph: MapGraph, agent: PlayerAgent, player_index: int) -> GameState:
      1. If player has 5+ cards, call force_trade_loop, get bonus armies
      2. Calculate base reinforcements via calculate_reinforcements
      3. Add trade bonus armies to total
      4. Apply territory bonus placements from trades (2 armies on owned territories from traded cards)
      5. Call agent.choose_reinforcement_placement with remaining armies
      6. Validate placements: all territories owned by player, armies sum to total
      7. Apply placements to state
      8. Set turn_phase to ATTACK, return new state

    - execute_attack_phase(state: GameState, map_graph: MapGraph, agent: PlayerAgent, player_index: int, rng: random.Random) -> tuple[GameState, bool]:
      Loop:
        action = agent.choose_attack(state) -- None means end phase
        If action: execute_attack, check if conquest, check if elimination (transfer cards, forced trade, check victory)
        If victory: return (state, True)
      After loop: if conquered_this_turn, draw_card
      Set turn_phase to FORTIFY, return (state, False)

    - execute_fortify_phase(state: GameState, map_graph: MapGraph, agent: PlayerAgent, player_index: int) -> GameState:
      action = agent.choose_fortify(state)
      If action: execute_fortify
      Return state

    - execute_turn(state: GameState, map_graph: MapGraph, agents: dict[int, PlayerAgent], rng: random.Random) -> tuple[GameState, bool]:
      player_index = state.current_player_index
      agent = agents[player_index]
      Reset conquered_this_turn=False, set turn_phase=REINFORCE
      Run reinforce -> attack -> fortify phases
      If victory during attack: return (state, True)
      Advance to next alive player, increment turn_number
      Return (state, False)

  <action>
    Create risk/game.py:

    - class RandomAgent: A simple agent that makes valid random moves using a seeded RNG.
      Implements all PlayerAgent protocol methods:
      - choose_reinforcement_placement: distribute armies randomly among owned territories
      - choose_attack: 70% chance to attack a random valid target, 30% chance to return None (end attacks)
      - choose_blitz: not used by default (use regular attacks for more granular testing)
      - choose_fortify: 50% chance to fortify between random connected territories, 50% skip
      - choose_card_trade: if forced, trade first valid set found; if not forced and has valid set, 50% chance to trade
      - choose_defender_dice: always roll max dice

    - run_game(map_graph: MapGraph, agents: dict[int, PlayerAgent], rng: random.Random, max_turns: int = 1000) -> GameState:
      1. Call setup_game(map_graph, len(agents), rng) to get initial state
      2. Initialize deck via create_deck, shuffle with rng, set on state
      3. Initialize empty card lists for each player
      4. Loop: execute_turn until victory or max_turns reached
      5. If max_turns reached without victory, raise RuntimeError("Game did not complete within max_turns")
      6. Return final state
