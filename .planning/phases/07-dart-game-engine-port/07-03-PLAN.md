CRITICAL: Python accesses `map_graph._continent_bonuses` directly (private). Dart cannot. Instead use `mapGraph.continentNames` (added in Plan 02) to iterate:

```dart
import 'dart:math';
import 'map_graph.dart';
import 'models/game_state.dart';

int calculateReinforcements(GameState state, MapGraph mapGraph, int playerIndex) {
  final playerTerritories = state.territories.entries
      .where((e) => e.value.owner == playerIndex)
      .map((e) => e.key)
      .toSet();

  final base = max(playerTerritories.length ~/ 3, 3);  // NOTE: ~/ not /

  int bonus = 0;
  for (final continent in mapGraph.continentNames) {
    if (mapGraph.controlsContinent(continent, playerTerritories)) {
      bonus += mapGraph.continentBonus(continent);
    }
  }
  return base + bonus;
}
```

validateFortify(GameState state, MapGraph mapGraph, FortifyAction action, int playerIndex):
