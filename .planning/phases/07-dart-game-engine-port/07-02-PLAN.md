}

Create `mobile/lib/engine/combat.dart`. Direct 1:1 translation of risk/engine/combat.py.

Critical translation rules (from research — enforce all):
1. `rng.nextInt(6) + 1` (NOT `rng.nextInt(6)`) — Python randint(1,6) is inclusive both ends
2. Sort descending: `..sort((a, b) => b.compareTo(a))` (NOT `..sort()` which is ascending)
3. Use `~/` for integer division — nowhere in combat, but set precedent
4. Territory map copies: `Map<String, TerritoryState>.of(state.territories)` then mutate then copyWith
5. Cards map uses String keys: `state.cards[playerIndex.toString()]`
6. Return Dart 3.x records: `(GameState, CombatResult, bool)` for executeAttack; `(GameState, List<CombatResult>, bool)` for executeBlitz
