6. Assert output matches expected values exactly

File path resolution in tests: use `Platform.script.resolve('fixtures/golden_combat.json').toFilePath()` or a relative path. Safer approach: use a helper that resolves relative to the test file directory. In flutter_test, use `Directory.current.path` + relative path since `flutter test` runs from the `mobile/` directory:

```dart
String fixturesPath(String filename) => 'test/engine/fixtures/$filename';
```

Structure per fixture group:

```dart
group('golden_combat', () {
  late List<dynamic> fixtures;

  setUpAll(() async {
    final raw = await File(fixturesPath('golden_combat.json')).readAsString();
    fixtures = json.decode(raw) as List<dynamic>;
  });

  for (final f in fixtures) {
    test(f['id'] as String, () {
      final inputState = GameState.fromJson(f['input_state'] as Map<String, dynamic>);
      final rolls = (f['injected_rolls'] as List).cast<int>();
      final rng = FakeRandom(rolls);
      final action = AttackAction(
        source: f['action']['source'] as String,
        target: f['action']['target'] as String,
        numDice: f['action']['num_dice'] as int,
      );
      // Need a minimal MapGraph — construct from the input_state territories
      // Use the actual classic.json for MapGraph (load once in setUpAll)
      final (newState, result, conquered) = executeAttack(inputState, mapGraph, action, 0, rng);
      expect(result.attackerLosses, f['expected_attacker_losses']);
      expect(result.defenderLosses, f['expected_defender_losses']);
      expect(conquered, f['expected_conquered']);
      // Deep equality on output_state
      final expectedState = GameState.fromJson(f['output_state'] as Map<String, dynamic>);
      expect(newState, expectedState);
    });
  }
});
```

MapGraph for tests: Load `mobile/assets/classic.json` using `File('assets/classic.json').readAsStringSync()` in setUpAll. This file exists from Phase 6.
