  <behavior>
    MediumAgent.chooseReinforcementPlacement:
    - Scores each continent by fraction of territories owned (owned / total)
    - Finds top continent by max(fraction, tiebreak: continentBonus)
    - Within top continent, finds player-owned border territories
    - Prefers external-facing borders (neighbor outside continent is enemy)
    - Places all armies on the weakest (fewest armies) qualifying territory
    - Fallback: any border territory with fewest armies
    - Fallback: random owned territory

    MediumAgent.chooseAttack:
    - Priority 1: territory-completing attacks (src.armies >= tgt.armies)
    - Priority 2: favorable attack (src.armies > tgt.armies) into top-scoring continent
    - Priority 3: blocking opponent continent completion (src.armies > tgt.armies)
    - Priority 4: any favorable attack (src.armies > tgt.armies)
    - Returns null when no candidates match any priority

    MediumAgent.chooseFortify:
    - Source: interior territories (all neighbors owned) with armies >= 2
    - Returns null if no interior territories exist
    - Returns null if no border territories exist
    - Source = interior territory with MOST armies
    - Moves armies_to_move = source.armies - 1
    - Target = reachable (via connectedTerritories) border with FEWEST armies

    MediumAgent.chooseCardTrade:
    - Identical to EasyAgent: returns first valid 3-card set or null

    MediumAgent.chooseAdvanceArmies:
    - Returns min_armies (conservative)

    HardAgent._borderSecurityRatio:
    - Returns sum(enemy_adjacent_armies) / max(own_armies, 1)
    - Higher = more vulnerable

    HardAgent._continentScores:
    - fraction = owned / total
    - score = fraction * continentBonus * continentProgressWeight
    - If fraction > 0.5: score *= 1.5 (exponential boost)

    HardAgent._opponentThreatScores:
    - For each alive non-self player: total_armies * 0.5 + continent_threat * threatWeight
    - continent_threat: bonus*2 if missing only 1 territory; bonus if >= 70% owned

    HardAgent.chooseReinforcementPlacement:
    - Score border territories by: BSR * borderSecurityWeight + continentScore
    - Concentrate on top 1-2: if armies <= 3 or only 1 border, put all on top
    - Otherwise: 2/3 on ranked[0], 1/3 on ranked[1]

    HardAgent.chooseAttack:
    - Guard: return null if no territory has armies >= 3 with armies > neighbor enemy
    - Priority 1: continent-completing attacks (src.armies >= tgt.armies)
    - Priority 2: block opponent continent (opponent owns all-but-2 in continent, src > tgt)
    - Priority 3: win_prob >= 0.6 attacks sorted by (win_prob * 2.0 + continent_value)
    - Priority 4: overwhelming force (src >= 3*tgt and src >= 4)
    - Returns null if none match

    HardAgent.chooseCardTrade:
    - Forced: call _bestTrade()
    - Not forced, < 3 cards: return null
    - Hold until cards.length >= 4 OR state.tradeCount >= 4
    - _bestTrade: prefer cards whose territory is owned by player (territory bonus)

    HardAgent.chooseAdvanceArmies:
    - source is interior (all neighbors owned post-conquest): return max_armies
    - target has 0 enemy neighbors, source has > 0: return min_armies
    - source has 0 enemy neighbors: return max_armies
    - Both border enemies: proportional split = int(max_armies * target_ratio), clamped [min, max]

    HardAgent.chooseFortify:
    - Same logic as MediumAgent for source selection (interior, most armies)
    - Target: reachable border with HIGHEST BSR (most vulnerable), not fewest armies
  </behavior>
