### Interaction Contracts

**SimulationControlBar**
- SegmentedButton with 3 segments: Slow (1000ms delay), Fast (200ms delay), Instant (0ms, batch turns)
- Default selection: Fast
- Play/Pause IconButton toggles simulation loop on/off
- Stop IconButton (destructive color) halts simulation and shows GameOverDialog with current leader or "Simulation Stopped" state
- Minimum touch target: 44px height for all buttons
- Layout: Single row, horizontally centered, 12px internal padding (component-specific, not from scale)

**TerritoryInspector**
- Triggered by tapping a territory on the map during simulation (or when paused)
- Displays as a compact Card widget anchored to bottom of map area (not a modal)
- Content: Territory name (heading), Owner name + color swatch, Army count, Continent name
- Dismissible by tapping elsewhere on map or tapping the same territory again
- Max width: 280px, centered horizontally

**SimulationStatusBar**
- Fixed height: 48px
- Content: "Turn {N}" left-aligned, "{PlayerName}'s turn" center-aligned with 12px color dot, phase label right-aligned
- Background: surfaceContainerHighest
- Text: bodyMedium for turn number, labelMedium for phase

**Speed Behavior**
- Slow: 1000ms between each bot turn, map updates visible per turn
- Fast: 200ms between each bot turn, map updates visible per turn
- Instant: No delay, batch all turns until game over, only final state rendered (with progress indicator)

## State Contracts

| State | Visual Treatment |
|-------|-----------------|
| Simulation running (Slow/Fast) | Play icon filled, speed segment highlighted, StatusBar updating per turn |
| Simulation paused | Pause icon switches to Play, StatusBar shows "Paused - Turn {N}", map remains interactive for inspection |
| Simulation running (Instant) | All controls disabled except Stop, StatusBar shows progress indicator + "Simulating... Turn {N}" |
| Simulation complete | GameOverDialog appears (reused), ControlBar disabled |
| Simulation error | Snackbar with error message, auto-pause, Stop button remains active |
| Territory inspected | TerritoryInspector Card visible at bottom of map, tapped territory highlighted with accent border |
