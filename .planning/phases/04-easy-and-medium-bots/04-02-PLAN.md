  <name>MediumAgent — continent-aware Risk bot</name>
  <files>risk/bots/medium.py, risk/game.py</files>
  <behavior>
**Constructor:**
- `__init__(self, rng: random.Random | None = None)`
- `self._rng = rng or random.Random()`
- `self._map_graph: MapGraph | None = None` (injected after construction, never passed to constructor)

**_continent_scores(state) -> dict[str, float]:**
- For each continent: score = owned_count / total_territories_in_continent
- owned = territories where ts.owner == state.current_player_index
- Tie-breaking at call site: prefer higher continent_bonus value

**_border_territories(state, owned) -> list[str]:**
- Returns owned territories that have at least one neighbor not owned by current player
- Uses self._map_graph.neighbors()

**choose_reinforcement_placement(state, armies) -> ReinforcePlacementAction:**
- Place ALL armies on a single target territory (not distributed — concentration is better strategy)
- Step 1: Score continents; find highest-scoring continent (prefer higher bonus on ties)
- Step 2: Find border territories in that continent (owned AND has enemy neighbor)
- Step 3: If found, place all armies on the border territory with fewest armies in that continent (most exposed)
- Step 4: If no border in top continent, try any border territory (any continent)
- Step 5: If no borders at all, place on random owned territory
- Returns ReinforcePlacementAction(placements={target: armies})

**choose_attack(state) -> AttackAction | None:**
- If self._map_graph is None: return None
- Build candidate list: all (source, target) pairs where source.owner == player, source.armies >= 2, target.owner != player
- If no candidates: return None
- Score each candidate:
  - Priority 1 (highest): Attacking completes a continent for the player (player owns all other territories in target's continent). Take even if disadvantaged by 1 army.
  - Priority 2: Target is in highest-scoring continent AND source.armies > target.armies (favorable odds)
  - Priority 3: Target is in a continent where an opponent owns all but one territory and that territory is the target (blocking). AND source.armies > target.armies.
  - Priority 4 (fallback): Any attack where source.armies > target.armies (favorable odds)
- Return AttackAction for the highest-priority candidate, or None if none qualify

**choose_blitz(state) -> None:**
- Always return None (same as RandomAgent; blitz differences are Hard bot territory)

**choose_fortify(state) -> FortifyAction | None:**
- If self._map_graph is None: return None
- Find interior territories: owned, armies >= 2, ALL neighbors are also owned by bot
- If no interior territories with surplus: return None
- Find border territories: owned, has at least one enemy neighbor
- If no border territories: return None
- Source = interior territory with most armies
- armies_to_move = source.armies - 1 (always leave at least 1)
- If armies_to_move < 1: return None
- Find reachable friendly territories from source via connected_territories()
- Target = reachable border territory with fewest armies (most exposed border)
- If no reachable border: return None
- Return FortifyAction(source=source, target=target, armies=armies_to_move)

**choose_card_trade(state, cards, forced) -> TradeCardsAction | None:**
- Identical to RandomAgent: always trade any valid set if one exists, regardless of forced
- Use is_valid_set() to find the first valid triple

**choose_defender_dice(state, territory, max_dice) -> int:**
- Always return max_dice (same as RandomAgent)

**Edge cases (must not crash):**
- _map_graph is None: all methods return safe fallback
- Bot owns 0 territories: choose_reinforcement_placement still called; all loops find no owned territories; fall through to random (will also be empty; return empty placements dict {} or raise — check what engine does with empty placement)
- All owned territories fully enclosed: fortify returns None
- All attacks unfavorable + no continent completion possible: attack returns None
