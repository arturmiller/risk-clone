Tools & Keyboard Shortcuts

Tools
✏️ Draw (D) — Linksklick setzt Vertices, schließt mit Klick auf Start-Vertex oder Doppelklick
↖ Select (V) — Vertices/Edges auswählen, verschieben, löschen (Del)
✋ Pan (Space) — Canvas verschieben, Mausrad = Zoom
🪣 Territory (T) — In Face klicken → Territorium erstellen, Name + Farbe vergeben
Shortcuts
Ctrl+Z — Undo
Ctrl+Y — Redo
Del — Ausgewähltes Element löschen
Ctrl+S — Speichern (JSON)
Ctrl+E — Export (Game-Format)
Escape — Tool abbrechen / Auswahl aufheben
